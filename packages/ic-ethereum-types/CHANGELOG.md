## [1.0.0] - 2024-09-30

### Added

- `ic_ethereum_types::Address`: An Ethereum address of 20 bytes.
- `ic_ethereum_types::serde_data`: Serialize and deserialize any hexadecimal type following Ethereum's format (prefixed with `0x`).
