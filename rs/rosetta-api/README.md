# Rosetta

## Running Tests Locally

### Prerequisites

Install [Just](https://github.com/casey/just#installation):
```bash
# macOS
brew install just

# Linux/macOS with Cargo
cargo install just

# Ubuntu/Debian
sudo apt install just
```

### Running Tests

```bash
# See all commands
just

# ICP tests
just test icp

# ICRC1 tests  
just test icrc1
```
